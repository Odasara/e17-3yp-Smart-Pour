Smart Pour-An automated coffee machine that is controlled by an app

Introduction

The Smart Pour is an improvement for traditional coffee makers. Traditional coffee making is inconvenient with today's hectic lifestyle. We, as a team, came up with a more convenient coffee making system that can be controlled by an app.
Main features of our system:

  • Schedule Coffe making
  • Ingredient tracking mechanism
  • Can be customized according to the user's preference
  • User-friendly interface
  • High security
  • Durability
  • Simplicity

Problem

To have a coffee with hectic life style and according to the personal preference
